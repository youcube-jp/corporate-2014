# youcube.jp

is built with [Middleman](http://middlemanapp.com/).

## 開発環境構築

```bash
$ bundle install --path vendor/bundle --jobs=4
```

## サーバー起動

```bash
$ bundle exec middleman server
```

http://localhost:4567/
